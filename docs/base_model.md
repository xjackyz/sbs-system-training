# BaseModel 类实现步骤

## 1. 初始化
- **`__init__` 方法**：接收 `config` 参数，初始化基础模型配置和设备设置。

## 2. 抽象方法
- **`forward` 方法**：抽象方法，子类必须实现的前向传播逻辑。

## 3. 模型保存和加载
- **`save` 方法**：保存模型状态和配置。
- **`load` 方法**：加载模型状态和配置。

## 4. 设备管理
- **`to_device` 方法**：将模型移动到指定设备。

## 5. 参数管理
- **`count_parameters` 方法**：统计模型参数数量。
- **`freeze` 方法**：冻结所有参数。
- **`unfreeze` 方法**：解冻所有参数。
- **`get_parameter_groups` 方法**：获取参数组。

## 6. 属性和描述
- **`num_parameters` 属性**：获取模型参数数量。
- **`__str__` 方法**：返回模型描述字符串。
